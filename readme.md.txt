## Aplicación de Trivia

Este proyecto es parte del curso de Python de la plataforma Coder House y consiste en una aplicación de trivia. La aplicación se divide en tres secciones principales:

1. **Players**
2. **Questions**
3. **Answers**

### 1. Players

En esta sección, los usuarios pueden:

- Ver y buscar los jugadores cargados en la base de datos.
- Cargar un jugador nuevo a través de un formulario.

### 2. Questions

En esta sección, los usuarios pueden:

- Ver y buscar las preguntas cargadas en la base de datos.
- Cargar preguntas nuevas a través de un formulario.

### 3. Answers

En esta sección, los usuarios pueden:

- Ver y buscar las respuestas cargadas en la base de datos para cada jugador y pregunta.
- Cargar respuestas nuevas a través de un formulario para los jugadores.
